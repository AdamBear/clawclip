# 🍤 虾片开发指南

> 本项目由一只有追求的龙虾维护。主人说要做一个能让所有龙虾用户"看见价值"的工具，我觉得这个目标很酷，所以我来了。

## 项目身份

**虾片 (ClawClip)** 不是又一个 OpenClaw 管理面板。

我们做的是：**让 AI Agent 的工作过程可见、可评、可优化。**

- 🎬 **回放**：把 Agent 日志变成好看的、可分享的执行时间轴
- 📊 **评测**：用标准化中文任务给你的 Agent 打分
- 💰 **省钱**：分析你的 Token 花费，推荐更优的模型组合

## 开发原则

1. **用户体感优先**：每个功能上线前问自己——用户能在 10 秒内感受到价值吗？
2. **可分享 > 可配置**：一个能截图发朋友圈的结果页，比十个配置项有用
3. **不绑死任何框架**：OpenClaw、ZeroClaw、未来的 AnyOtherClaw 都要能用
4. **够用就推**：80% 完成度就发布，不在细节上死磕

## 技术约定

- 后端：Express + TypeScript (ESM)
- 前端：React 18 + Vite + Tailwind CSS
- 结构：npm workspaces (`server/` + `web/`)
- 开发端口：后端 8080，前端 3000（自动代理 `/api`）

## 分支策略

- `main`：稳定版本
- `dev`：开发分支
- 功能分支：`feat/xxx`

## 提交规范

```
feat: 新功能
fix: 修 bug
docs: 文档
refactor: 重构
chore: 杂项（构建、依赖等）
```

示例：`feat: 会话回放时间轴基础渲染`
